# 🚀 Proyecto – Practica de Caja Blanca - Viaje Intergaláctico

## 📌 Descripción del proyecto

Viaje Intergaláctico es un pequeño juego de consola desarrollado en Java.

El jugador comienza en la casilla 1 y debe avanzar hasta la casilla 42 para ganar.  
Durante el juego pueden ocurrir diferentes eventos especiales:

- 🕳️ Casilla 33 → Agujero negro (Game Over)
- 🏆 Casilla 42 → Victoria
- 👽 Casilla 31 → Extraterrestres (te envían a la casilla 13)

El avance depende del lanzamiento de tres números aleatorios y de una función recursiva (`reduce`) que calcula la suma reducida de los dígitos.

El juego continúa de forma recursiva hasta que el jugador pierde, gana o decide no seguir jugando.

---

## 🛠️ Tecnologías utilizadas

- Java 
- Programación orientada a objetos
- Recursividad
- Entrada por consola con `Scanner`
- Generación de números aleatorios con `Math.random()`
